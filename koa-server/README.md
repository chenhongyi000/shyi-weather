# 聊天室

## 技术栈

- koa2
- mongoose
- ejs
- cors
- jsonwebtoken
- bcryptjs
- crypto-js

# 安装

```bash
npm install
```
# 启动

```bash
npm run dev
```
# 访问

```bash
http://localhost:3000
```
# 登录

```bash
http://localhost:3000/login
```
# 注册

```bash
http://localhost:3000/register
```
# 聊天室

```bash
http://localhost:3000/chat
```
